瓶塞参数化
=========

所有测量单位均为毫米。

主要参数
-------

- 柄部直径：它应该是瓶颈的内径或略小一些。
- 旋钮直径：旋钮的直径。它必须大于柄部直径，出于美观考虑，应略大于瓶颈的外径。

可选参数：
--------

- 柄部直径增加：在柄部内端增加柄部直径的量（如香槟软木塞）。柄部直径和柄部直径增加之和应略大于瓶颈。当使用TPU 95A时，大多数情况下1的默认值就足够了。当使用更软的材料时，您需要增加此值以确保紧密贴合。
- 柄部长度：柄部的长度。通常情况下，20就足够了。
- 柄部倒角：柄部内端的倒角，使瓶塞更容易插入。2或3通常是不错的数值。
- 旋钮长度：旋钮的长度。15-20对于大多数情况来说都是不错的数值。
- 上旋钮倒角：旋钮顶部的倒角。仅用于美观。较大的倒角使其更像蘑菇，较小的倒角更像圆柱体。
- 下旋钮倒角：旋钮下侧的倒角。它应该是1或2，以提高用户拔出瓶塞时的抓握力。
