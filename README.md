3D 打印注意事项
-------------

1. 保持耗材干燥。吸了水的耗材打印会拉丝，回抽都没用。不常用的耗材打印前最好用干燥盒烘干，效果明显。
2. 注意耗材的参数，尤其是温度和速度。不要超出耗材的最大速度。**慢工出细活。**

Tips 小贴士
----------

1. 表面拉丝使用热风枪吹一下（或用打火机快速燎一下）可以快速解决，比用刀片刮削方便。
2. 锋利边缘也可以使用热风枪软化，再在桌子等坚硬表面按压一下。
