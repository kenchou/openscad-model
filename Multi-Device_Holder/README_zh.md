多设备支架
========
[[English Readme (README.md)]]

多设备支架，可以架在多口充电器上。可定制多部手机、平板、充电宝等设备。

定制参数:
```
baseWidth       = 底座长
baseDeep        = 底座宽
baseHeight      = 底座厚度
separatorWidth  = 隔板厚度
separatorDeep   = 隔板宽
separatorHeight = 隔板高
rechargerHeight = 充电器高
rechargerWidth  = 充电器宽。由于cuboid是外部宽度，此处实际应设为：充电器实际宽+壁厚baseHeight，建议留足够空间。
```
